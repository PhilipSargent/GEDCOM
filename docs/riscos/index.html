!Family is a Genealogy database (GEDCOM) editor and browser for the RISC OS desktop

This screenbook describes !Family version 2.2.7, though the graphics include the odd screenshot from !Family 2.24, and XFamily 0.11. Some features are mentioned which are not yet implemented, and the !Family user interface is evolving towards that of XFamily (hence the mix of screenshots). This will all get less confusing as the interfaces stabilise over the next few months.

If you are migrating from !Family 2.24 (the last version released as "stable"), please read Appendix A, as a small utility program needs to be run on your existing GEDCOM file(s) before loading into version 2.25, 2.26 or this current release 2.2.7
