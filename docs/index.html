The X!Family Documentation

Loosely Based on !Family 2.18 for RISC OS by Dennis Howes,
now developed in parallel. The User interface of !Family v2.27 for RISC OS most closely resembles current versions ofxFamily v0.2.2

X!Family is a Genealogy database (GEDCOM) editor and browser for the X desktop. It implements version 5.3 of GEDCOM, but respects some syntax from earlier versions of the GEDCOM standard.
