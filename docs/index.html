The X!Family Documentation

Loosely Based on !Family 2.18 for RISC OS by Dennis Howes, then developed in parallel until about 2003.

The User interface of !Family v2.27 for RISC OS and xFamily v0.2.2 for Linux closely resemble each other.

!Family and xFamily are Genealogy database (GEDCOM) editors and browsers. They implement version 5.3 of GEDCOM, but respect some syntax from earlier versions of the GEDCOM standard. See the GEDCOM appendices Linux and RISCOS GEDCOM documentation details.
