GEDCOM 5.5.5 Primitive Elements (Page 80)

{Size=4:35} DATE:= [ <DATE_CALENDAR> | <DATE_CALENDAR_ESCAPE> + space + <DATE_CALENDAR> ] where: space = U+0020, the space character.
{Size=8:39} DATE_APPROXIMATED:= [ ABT + space + <DATE> | CAL + space + <DATE> | EST + space + <DATE> ] where: ABT = About, meaning the date is not exact. CAL = Calculated mathematically, for example, from an event date and age. EST = Estimated.
{Size=4:35} DATE_CALENDAR:= [ <DATE_GREG> | <DATE_JULN> | <DATE_HEBR> | <DATE_FREN> ] The selection is based on the <DATE_CALENDAR_ESCAPE> that precedes the <DATE_CALENDAR> value immediately to the left. If no escape is present, @#DGREGORIAN@ is assumed. [cite: 3455, 3456]
{Size=9:11} DATE_CALENDAR_ESCAPE:= [ @#DHEBREW@ | @#DFRENCH R@ | @#DGREGORIAN@ | @#DJULIAN@ | @#DUNKNOWN@ ] The escape sequence used to specify the calendar used for the date. Note: @#DGREGORIAN@ is the default if no escape is present.
{Size=4:35} DATE_GREG:= [ <YEAR> | <MONTH> + space + <YEAR> | <DAY> + space + <MONTH> + space + <YEAR> ] Gregorian calendar date. One- and two-digit years are illegal; they must be padded to at least 3 digits (e.g., 0095).
{Size=4:35} DATE_JULN:= [ <YEAR> | <MONTH> + space + <YEAR> | <DAY> + space + <MONTH> + space + <YEAR> | <DAY> + space + <MONTH> + space + <DUAL_STYLE_YEAR> ] Julian calendar date. GEDCOM 5.5.5 allows dual-style years for the Julian calendar.
{Size=4:35} DATE_HEBR:= [ <YEAR> | <MONTH_HEBR> + space + <YEAR> | <DAY> + space + <MONTH_HEBR> + space + <YEAR> ] Hebrew calendar date.
{Size=4:35} DATE_FREN:= [ <YEAR> | <MONTH_FREN> + space + <YEAR> | <DAY> + space + <MONTH_FREN> + space + <YEAR> ] French Revolutionary calendar date.
{Size=7:9} DUAL_STYLE_YEAR:= <YEAR> + / + <DIGIT><DIGIT> Used for dates during the transition between the Julian and Gregorian calendars, where the second year is indicated by its last two digits (e.g., 1751/52).
{Size=1:255} DATE_VALUE:= [ <DATE> | <DATE_PERIOD> | <DATE_RANGE> | <DATE_APPROXIMATED> | <DATE_INT> | <DATE_PHRASE> ] A general container for any valid date representation, including ranges, periods, and approximated dates.
